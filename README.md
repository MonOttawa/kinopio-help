# kinopio-meta

A small, simple static website for pages like:
- Privacy Policy
- Terms of Service
- API Docs

still a wip, using
https://www.11ty.io/docs/usage/
to be served from meta.kinopio.club

## Install

git clone https://github.com/kinopio-club/kinopio-meta.git
cd kinopio-meta
npm install

## Run

npx @11ty/eleventy --serve --port=8081
http://localhost:8081
