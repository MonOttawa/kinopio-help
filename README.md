# Kinopio Help

help.kinopio.club

An [11ty](https://www.11ty.io/docs/usage/) website for Kinopio help and info pages, such as

- support articles
- api docs
- privacy policy
- terms of service

## Setup

	$ npm install
	$ npm install stylus --global
	$ npm install nodemon --global

## Run

	$ npm run start
	$ npm run styles

http://localhost:8081
