# kinopio-meta

A simple static website for Kinopio meta pages.

still a wip, using
https://www.11ty.io/docs/usage/
to be served from meta.kinopio.club

## Install

git clone https://github.com/kinopio-club/kinopio-meta.git
cd kinopio-meta
npm install

## Run

npm run serve
http://localhost:8081
